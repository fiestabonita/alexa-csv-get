# alexa-csv-get
gets latest alexa toplist in nodejs, bash, and php

NodeJS (Coooooool!)
```
sudo apt-get install nodejs node-request npm
npm install unzip
npm install csv2
nodejs alexa.js > topsites.json
```

Bash Script (Nice!)
``` bash alexa.sh ```

PHP (Could be prettier for sure)
``` sudo apt-get install php5-cli ```
``` php alexa.php ```
